# Введение в дипломную работу
## Дипломное проектирование, 4 курс

### Актуальность темы

В современном мире информационных технологий особенно важной становится задача создания эффективных и удобных веб-приложений.

### Цель работы

Разработка веб-портфолио программиста с использованием современных технологий.

### Задачи исследования

1. Проанализировать существующие решения
2. Выбрать оптимальный технологический стек
3. Разработать функциональное веб-приложение
4. Провести тестирование и отладку
